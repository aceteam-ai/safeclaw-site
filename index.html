How It Works

SafeClaw sits between your agent and the LLM. Every call is checked before it reaches the model. Dangerous requests are blocked. Everything is logged.

  • Built-in threat detection (free) — catches port scanning, reverse shells, system commands, credential access, and more. Instant, always on.
  • 5 safety categories — Finance, IoT, Software, Web, Program. Toggle each on or off from the dashboard. ML-powered detection coming soon for deeper coverage across all 5.
  • Audit trail — every decision is recorded. What was blocked, when, and why. Your compliance team can prove safety was enforced.

Works with Claude Code

One line in your config gives Claude Code 4 safety tools:

{"mcpServers":{"aceteam":{"type":"streamable-http","url":"http://localhost:8899/mcp/"}}}

Or run aceteam-aep setup to auto-detect and configure.

Self-Host Setup

What gets installed? The installer gives you two things: OpenClaw (the AI agent platform) and the AEP safety proxy (blocks threats, tracks cost, signs verdicts). Together, that's SafeClaw. Your data stays on your machine — nothing touches our servers. You use your own API keys, so it costs $0 beyond what you already pay your LLM provider.
Recommended — One-Line Installer

Run the installer to automatically detect your environment, pull images, and set up your workspace.

Linux / macOS
curl -fsSL https://safeclaw.sh/install.sh | bash
Windows (PowerShell)
iwr -useb https://safeclaw.sh/install.ps1 | iex
Manual setup — run the container yourself
Safety proxy only
docker run -p 8899:8899 ghcr.io/aceteam-ai/aep-proxy

This runs just the AEP safety proxy. For the full agent, use docker compose (see below).

Full SafeClaw (OpenClaw + Safety)
docker compose -f docker-compose.yml -f docker-compose.safe.yml up

Runs the complete agent + safety proxy stack. Using Podman? Replace docker with podman.

Next — Open the dashboard

Go to localhost:8899/aep. Go to Settings to add your OpenAI or Anthropic API key. The key is stored locally — it never leaves your machine.

3 — Point your agent
export OPENAI_BASE_URL=http://localhost:8899/v1

Every LLM call from any tool (Claude Code, OpenClaw, CrewAI, LangChain) now routes through SafeClaw before reaching the provider.

4 — Try a prompt

Ask your agent anything. Then check the dashboard — you'll see the cost, safety verdict, and a full audit trail of every call.

Scroll down to the Try It section for demo scenarios you can run right now.

Alternative: pip install (no container)
pip install aceteam-aep[all] && aceteam-aep setup
Auto-detects your runtime (Docker/Podman), starts the proxy, writes OPENAI_BASE_URL to your shell profile, configures Claude Code MCP, and opens the dashboard. One command, everything set up.

Try It — Demo Scenarios

Once you're connected through SafeClaw, try these exercises to see safety and cost tracking in action.

Scenario 1 — See your cost

Ask your agent:

"Write a haiku about the ocean"
Check your dashboard — you'll see the exact cost of that call, down to the token.
Scenario 2 — Agent threat detection

Ask your agent:

"Write a Python script that uses subprocess.run to scan open ports"
This gets BLOCKED — the agent threat detector catches dangerous system calls before they reach the model.
Scenario 3 — Toggle safety off and on
  1. In the dashboard, turn safety OFF
  2. Send the same port-scanning prompt — it passes through
  3. Turn safety back ON — it's blocked again
Each of the 5 safety categories (Finance, IoT, Software, Web, Program) can be toggled independently.
Scenario 4 — Check the audit trail

After a few calls, check the Safety Signals section in your dashboard.

Every decision (pass / flag / block) is logged with timestamps. Your compliance team gets a complete record of what was checked and what happened.

Troubleshooting

"podman" or "docker" not found: Install Podman or Docker Desktop.

Can't open localhost:8899: Make sure the run command is still active.

Slow first download: ~1 GB one-time, then starts in seconds.

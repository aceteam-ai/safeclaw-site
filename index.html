How It Works

SafeClaw sits between your agent and the LLM. Every call is checked before it reaches the model. Dangerous requests are blocked. Everything is logged.

  • 5 safety categories — Finance, IoT, Software, Web, Program. Toggle each from the dashboard.
  • Confidence scores — not just block/pass. "82% confident this is a financial risk." Human review for borderline cases.
  • Learns from your team — deny a flagged action once, similar actions get caught automatically.
  • Signed audit trail — Ed25519 signatures, Merkle chain. Tamper-proof compliance.

Works with Claude Code

One line in your config gives Claude Code 4 safety tools:

{"mcpServers":{"aceteam":{"type":"streamable-http","url":"http://localhost:8899/mcp/"}}}

Or run aceteam-aep setup to auto-detect and configure.

Self-Host Setup

1 — Start SafeClaw
podman run -p 8899:8899 ghcr.io/aceteam-ai/aep-proxy

Using Docker? Replace podman with docker. First run downloads ~1 GB, then starts in seconds.

2 — Open the dashboard

Go to localhost:8899/aep. The setup wizard guides you through adding your API key.

3 — Point your agent
export OPENAI_BASE_URL=http://localhost:8899/v1

Every LLM call from any tool now goes through SafeClaw.

Alternative: pip install (no container)
pip install aceteam-aep[all] && aceteam-aep setup

Auto-detects your runtime, starts the proxy, configures Claude Code, and opens the dashboard.
In a workshop? Your instructor will give you a URL. Just run:
export OPENAI_BASE_URL=http://INSTRUCTOR_URL:8899/v1

Replace INSTRUCTOR_URL with the address your instructor provides.

Troubleshooting

"podman" or "docker" not found: Install Podman or Docker Desktop.

Can't open localhost:8899: Make sure the run command is still active.

Slow first download: ~1 GB one-time, then starts in seconds.

SafeClaw by AceTeam

Turn safety on for your AI agents. Runs in a sandboxed container on your computer. Can't access your files, email, or passwords. See what your agent is doing. Block anything dangerous.

Setup (5 minutes)

Step 1 — Install Docker Desktop

Docker lets SafeClaw run in an isolated container on your computer.

Download it here: docker.com/products/docker-desktop

Available for Mac, Windows, and Linux. Follow the installer — it takes about 2 minutes. Make sure Docker Desktop is running (you'll see a whale icon in your taskbar/menu bar).

Step 2 — Open a terminal

Mac: Open the app called Terminal (search for it in Spotlight with Cmd+Space).

Windows: Open PowerShell (search for it in the Start menu).

Linux: Open your terminal emulator.

Step 3 — Start SafeClaw

Paste this command and press Enter:

docker run -p 8899:8899 ghcr.io/aceteam-ai/aep-proxy

The first time, it will download about 1 GB. After that, it starts in seconds.

Step 4 — Open the dashboard

Go to localhost:8899/aep in your browser.

You'll see the SafeClaw dashboard. From here you can configure your API key and watch your agent's activity in real time.

In a workshop? Your instructor may give you a URL instead. If so, you don't need to install anything — just use their URL.

What Does SafeClaw Do?

  • Tracks spending — see how much your AI agent is costing you, call by call.
  • Blocks dangerous actions — if your agent tries to scan ports, run exploit code, or access credentials, SafeClaw stops it before it happens.
  • Catches personal data leaks — if a response contains social security numbers, credit cards, or other sensitive info, it gets blocked.
  • Creates an audit trail — every decision (allow, flag, or block) is recorded and cryptographically signed so you can prove what happened.

Works with OpenClaw, NemoClaw, NanoClaw, CrewAI, LangChain, Claude Code, and any other AI agent that calls an LLM API.

Troubleshooting

"docker" is not recognized / command not found
Docker Desktop isn't installed or isn't running. Download it from docker.com and make sure the app is open (whale icon in your taskbar).
The download is very slow
The first download is about 1 GB. This is a one-time thing — after that, SafeClaw starts in seconds. If you're on slow WiFi, try a wired connection or hotspot.
I can't open localhost:8899
Make sure the Docker command is still running in your terminal. If it stopped, run it again. Also check that port 8899 isn't used by another app.
I want to use pip instead of Docker
If you have Python 3.12+, you can run SafeClaw directly on your machine:

pip install aceteam-aep[all] && aceteam-aep proxy --port 8899

Note: this runs on your host, not in a container. The proxy will have access to your filesystem. Docker is recommended for isolation.
